# Mango E-commerce Website 🥭

A modern, responsive e-commerce website for premium mango varieties built with React and Vite.

## Features

- 🥭 Browse premium mango varieties (Alphonso, Kesar, Dasheri, Langra, Banganapalli, Totapuri)
- 📊 Interactive dashboard with seasonal trends and availability charts
- 🔍 Filter mangoes by season, taste profile, and origin
- 📝 Order inquiry form with validation
- 📱 Fully responsive design
- 🎨 Beautiful mango-inspired color palette

## Tech Stack

- **React** - UI library
- **Vite** - Build tool
- **React Router** - Routing
- **TanStack Query** - Data fetching
- **Tailwind CSS** - Styling
- **Recharts** - Data visualization
- **React Hook Form** - Form handling
- **React Hot Toast** - Notifications
- **date-fns** - Date utilities

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Nitinkumargits/Mango-.git
cd mango
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
